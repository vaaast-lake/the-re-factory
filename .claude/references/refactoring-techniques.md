-   **Composing Methods** (메서드 구성)
    -   **Extract Method** (메서드 추출: 코드 조각을 별도 메서드로 분리)
    -   **Inline Method** (메서드 인라인화: 단순한 메서드를 호출부에 직접 삽입)
    -   **Extract Variable** (변수 추출: 복잡한 표현식을 변수로 분리)
    -   **Inline Temp** (임시 변수 인라인화: 단순 임시 변수를 표현식으로 대체)
    -   **Replace Temp with Query** (임시 변수를 질의 메서드로 전환)
    -   **Split Temporary Variable** (임시 변수 분리: 여러 용도로 쓰이는 변수를 각각 분리)
    -   **Remove Assignments to Parameters** (매개변수 대입 제거)
    -   **Replace Method with Method Object** (메서드를 메서드 객체로 전환)
    -   **Substitute Algorithm** (알고리즘 교체: 더 명확한 알고리즘으로 대체)
-   **Moving Features Between Objects** (객체 간 기능 이동)
    -   **Move Method** (메서드 이동: 메서드를 더 적합한 클래스로 이동)
    -   **Move Field** (필드 이동: 필드를 더 적합한 클래스로 이동)
    -   **Extract Class** (클래스 추출: 두 가지 일을 하는 클래스를 둘로 분리)
    -   **Inline Class** (클래스 인라인화: 역할이 미미한 클래스를 다른 클래스에 병합)
    -   **Hide Delegate** (위임 숨기기: 중간 객체로 위임 관계 감추기)
    -   **Remove Middle Man** (중개자 제거: 과도한 위임 제거)
    -   **Introduce Foreign Method** (외래 메서드 도입: 수정 불가능한 클래스에 메서드 추가)
    -   **Introduce Local Extension** (지역 확장 도입: 서브클래스나 래퍼로 기능 확장)
-   **Organizing Data** (데이터 구성)
    -   **Self Encapsulate Field** (필드 자체 캡슐화: 직접 접근을 getter/setter로 변경)
    -   **Change Value to Reference** (값을 참조로 전환)
    -   **Change Reference to Value** (참조를 값으로 전환)
    -   **Replace Array with Object** (배열을 객체로 전환: 각 요소가 다른 의미를 가질 때)
    -   **Duplicate Observed Data** (관찰 대상 데이터 복제: GUI-비즈니스 로직 분리)
    -   **Change Unidirectional Association to Bidirectional** (단방향 연관을 양방향으로)
    -   **Change Bidirectional Association to Unidirectional** (양방향 연관을 단방향으로)
    -   **Replace Magic Number with Symbolic Constant** (매직 넘버를 상징적 상수로)
    -   **Encapsulate Field** (필드 캡슐화: public 필드를 private으로)
    -   **Encapsulate Collection** (컬렉션 캡슐화: 컬렉션 직접 반환 대신 읽기 전용 뷰 제공)
    -   **Replace Type Code with Class/Subclasses/State-Strategy** (타입 코드를 클래스/서브클래스/상태-전략으로)
    -   **Replace Subclass with Fields** (서브클래스를 필드로: 상수만 다른 서브클래스 제거)
-   **Simplifying Conditional Expressions** (조건식 단순화)
    -   **Decompose Conditional** (조건식 분해: 복잡한 조건식을 메서드로 추출)
    -   **Consolidate Conditional Expression** (중복 조건식 통합)
    -   **Consolidate Duplicate Conditional Fragments** (중복 조건 코드 통합)
    -   **Remove Control Flag** (제어 플래그 제거: break/return으로 대체)
    -   **Replace Nested Conditional with Guard Clauses** (중첩 조건문을 보호 구문으로)
    -   **Replace Conditional with Polymorphism** (조건문을 다형성으로 전환)
    -   **Introduce Null Object** (Null 객체 도입: null 체크를 객체로 대체)
    -   **Introduce Assertion** (단언 도입: 전제 조건을 명시적으로 표현)
-   **Simplifying Method Calls** (메서드 호출 단순화)
    -   **Rename Method** (메서드 이름 변경: 의도를 명확히 드러내는 이름으로)
    -   **Add Parameter** (매개변수 추가)
    -   **Remove Parameter** (매개변수 제거: 사용하지 않는 매개변수 삭제)
    -   **Separate Query from Modifier** (질의와 변경 분리: 조회와 수정을 별도 메서드로)
    -   **Parameterize Method** (메서드 매개변수화: 비슷한 메서드를 하나로 통합)
    -   **Replace Parameter with Explicit Methods** (매개변수를 명시적 메서드로)
    -   **Preserve Whole Object** (객체 전체 전달: 여러 필드 대신 객체 자체를 전달)
    -   **Replace Parameter with Method Call** (매개변수를 메서드 호출로: 메서드가 직접 조회 가능한 경우)
    -   **Introduce Parameter Object** (매개변수 객체 도입: 여러 매개변수를 객체로 묶기)
    -   **Remove Setting Method** (설정 메서드 제거: 변경 불가능한 필드의 setter 삭제)
    -   **Hide Method** (메서드 숨기기: 외부에서 사용하지 않는 메서드를 private으로)
    -   **Replace Constructor with Factory Method** (생성자를 팩토리 메서드로)
    -   **Replace Error Code with Exception** (오류 코드를 예외로)
    -   **Replace Exception with Test** (예외를 테스트로: 예상 가능한 조건은 if문으로)
-   **Dealing with Generalization** (일반화 처리)
    -   **Pull Up Method/Field** (메서드/필드 상향: 중복 기능을 부모 클래스로)
    -   **Push Down Method/Field** (메서드/필드 하향: 일부만 사용하는 기능을 자식으로)
    -   **Pull Up Constructor Body** (생성자 본문 상향)
    -   **Extract Superclass** (상위 클래스 추출: 공통 기능을 부모 클래스로)
    -   **Extract Interface** (인터페이스 추출: 공통 인터페이스 정의)
    -   **Extract Subclass** (하위 클래스 추출: 일부 인스턴스만의 특성 분리)
    -   **Collapse Hierarchy** (계층 병합: 거의 같은 부모-자식 클래스 통합)
    -   **Form Template Method** (템플릿 메서드 형성: 비슷한 알고리즘을 템플릿으로)
    -   **Replace Inheritance with Delegation** (상속을 위임으로)
    -   **Replace Delegation with Inheritance** (위임을 상속으로)
